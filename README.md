# 初中词汇超级单词表

一个可体验的 H5 英语词汇学习 Demo，面向初中生词汇听说读写训练。

## 在线体验

GitHub Pages 发布后访问仓库 Pages 地址。

- 正式入口：`index.html`
- Demo 入口：`demo.html`

Demo 会自动预置：

- 外研版 八年级上
- 已导入词表
- 每日计划 2 个词
- 今日新学、速刷复习、巩固复习、深度复习任务

## 功能

- 选择单词书
- 导入词表
- 制定每日计划
- 艾宾浩斯复习任务
- 七步学习：听、学、读、译、拆分、拼读、拼写
- 英式 IPA / 美式 IPA
- 浏览器发音播放

## 微信小程序版

`wechat-miniprogram/` 内提供微信小程序工程骨架，包含登录注册、学习进度同步、活跃度埋点和数据看板云函数。

部署说明见 [DEPLOY_WECHAT.md](./DEPLOY_WECHAT.md)。
